Food Ordering System

Description:
A web-based food ordering system built using Django.

Features:
- User Registration and Login
- Menu Display
- Add to Cart
- Checkout System
- Order History
- Admin Dashboard
- Order Filtering
- Pagination
- Sales Statistics Charts

Technologies Used:
- Python
- Django
- SQLite
- Bootstrap
- Chart.js

How to Run:
1. Install dependencies:
   pip install -r requirements.txt

2. Run server:
   python manage.py runserver
